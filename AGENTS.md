# AGENTS.md — KBD-WSI

## Project
Whole Slide Imaging (WSI) research. Three planned modules:

- **Compression** — vector-quantize small patches of WSI; compress + extract robust representations
- **Representation** — feature extraction from WSIs; bench + data curation submodules
- **Generation** — text generation from WSI representations; training scripts for open-weight + proprietary LMs; data curation schema

No code, config, or dependencies exist yet. Only this README-authored vision and two commits on `main`.
